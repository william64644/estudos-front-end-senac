Atividade 11

1) A alternativa correta para cada pergunta.
2) A alternativa que o usuário escolheu para cada pergunta.
3) A soma total de acertos (quantidade de respostas corretas).

Atividade 15

Atividade 24

Crie uma calculadora básica com interface em HTML e funcionalidades em JavaScript. O layout da calculadora é definido em um arquivo CSS externo. As funcionalidades incluem operações aritméticas básicas, cálculo de quadrado, raiz quadrada e manipulação de entrada (apagar, limpar). O botão "=" executa a expressão matemática usando eval(). Além disso, a calculadora altera a cor de fundo da página para tons aleatórios ao passar o mouse sobre o botão "=" e reseta a cor ao clicar no botão "⌫/C".

Clique aqui

Atividade 25

Crie um documento HTML que, ao pressionar uma tecla, altera a cor de fundo da página para uma cor específica com base na tecla pressionada. A cor é definida em um switch e cada tecla adiciona um texto à página com a cor correspondente. A função fundoBranco() define o fundo da página como branco quando a tecla é liberada, e um alerta é exibido ao carregar a página pedindo para que o usuário escreva algo.

Clique aqui
